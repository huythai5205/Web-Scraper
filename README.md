# Web-Scraper

Scrapes 'The Verge' website for articles. A user can save articles they like. Saved articles are saved in a Mongo database. A user can then add note for the article and remove from the saved list.

## Technologies Used:

### Front-end:
* AngularJs 5
* BootStrap

### Back-end:
* NodeJs
* ExpressJs
* MongoDb
* Mongoose
* cheerio
