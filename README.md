# Web-Scraper

Scrapes 'The Verge' website for articles.  User can save articles they likes.  Saved articles are save in a mongo database.  User can then add note for the arcticle and remove from the saved list.

## Technologies Used:

### Front-end:
* AngularJs 5
* BootStrap

### Back-end:
* NodeJs
* ExpressJs
* MongoDb
* Mongoose
* cheerio
